# MD General Medicine — Theory Paper & Answer Template

> What an MD Medicine theory paper looks like, and how to write an answer sheet that banks marks.
> Verified against the NMC pattern + published university schemes (TNMGRMU, BFUHS). The internal split
> (number of LAQs vs short notes per paper) varies by university — the **structure and the answer
> discipline** below are stable; the exact question count is not guaranteed.

## What each paper looks like

| | |
|---|---|
| Papers | **4**, written on separate days |
| Per paper | **100 marks · 3 hours** |
| Question types | **LAQ** (long-answer essay, ~10–15 marks) + **SAQ** (short note, ~3–5 marks) |
| Typical shape | a small number of LAQs (often with internal *OR* choice) + a larger set of short notes |
| Some papers add | a "diagnostic / therapeutic dilemma" SAQ (a short applied scenario) |

> A representative 100-mark paper might run ~2–4 LAQs (with choice) + ~6–10 short notes. **Attempt every
> question** — short notes give partial credit; a blank scores nothing.

## How an LAQ is marked (step / block marking)

An examiner marks an LAQ as **independent mark blocks**, not as flowing prose. You earn the marks for each
block you produce, even if another block is blank — exactly like CBSE step-marking, re-skinned for medicine.
Structure every LAQ under headed sub-sections:

| Block | What scores | Typical weight (10–15m LAQ) |
|-------|-------------|------------------------------|
| **Definition / intro** | correct definition, key epidemiology | 1–2 |
| **Aetiopathogenesis** | causes + mechanism (each a sub-point) | 2–3 |
| **Clinical features** | symptoms / signs as a checklist | 2–3 |
| **Investigations** | relevant tests + expected findings | 2–3 |
| **Management** | conservative / medical / surgical, doses where relevant | 2–4 |

- **Tables, flowcharts and labelled diagrams carry independent marks** — a classification table, a
  management algorithm, or a labelled sketch (e.g. malaria life-cycle, ascending-paralysis pattern,
  an ECG tracing) scores on its own. Draw / tabulate whenever the answer calls for it.
- The high-value, examiner-favourite lines (e.g. the *life-threatening complication* and its *monitoring*,
  the *first-line drug + dose*) are where the marks concentrate — lead with them.

## How an SAQ (short note) is marked

2–5 mark blocks: **definition (1) + key-points checklist (1–2) + one classification / criterion / clinical
pearl (1)**. Same independent-block scoring. Be enumerative and compact — short notes reward coverage, not
prose.

## Writing the answer sheet (exam-hall order)

1. **Read the whole paper first**; mark which LAQ *OR*-choice you'll take (prepare only the side you know).
2. **Answer LAQs with headed sub-sections** (Defn / Aetiopath / CF / Ix / Mx) — the structure *itself* banks
   marks and makes the examiner's job easy.
3. **Always draw the table / flowchart / labelled diagram** where one is asked — independent marks.
4. **Attempt every short note** — partial credit is real; never leave one blank.
5. **Don't overrun one LAQ and starve a paper.** Budget time by marks; a paper below 40% fails the whole
   theory head regardless of how strong the others are.
6. Write legibly — content is marked separately from presentation, but an illegible answer earns nothing.

## Passing criteria ⚠️ two constraints, both binding (no grace)

1. **Theory:** ≥ **40% in EACH of the four papers** AND ≥ **50% aggregate** (≥ 200 / 400).
2. **Practical / clinical + viva:** separate ≥ **50%** gate — assessed independently, **out of scope here**.

Failing either head = fail, even if the combined total clears 50%.
