## Key derivations / must-know results
- Mass defect `Delta m = [Z m_p + (A-Z) m_n] - M`; binding energy `E_b = Delta m c^2 = Delta m(u) x 931.5 MeV` (Q9).
- Nuclear density `rho = m / ((4/3) pi R0^3) ≈ 2.3 x 10^17 kg/m^3`, independent of A (Q17).
- Decay law `-dN/dt = lambda N => N = N0 e^(-lambda t)` (Q18).
- `T_(1/2) = 0.693/lambda`; `tau = 1/lambda`; `T_(1/2) = 0.693 tau` (Q14, Q19).
- Activity `R = lambda N = R0 e^(-lambda t)`; SI unit becquerel (Bq); `1 Ci = 3.7 x 10^10 Bq` (Q15).
- `R = R0 A^(1/3)` with `R0 = 1.2 fm`; radius ratio `(A1/A2)^(1/3)` (Q25).
- Q-value `Q = (m_reactants - m_products) c^2` (Q26, Q27).
- `1 u = 931.5 MeV/c^2`; per fission ²³⁵U ≈ 200 MeV; D-T fusion ≈ 17.6 MeV.
