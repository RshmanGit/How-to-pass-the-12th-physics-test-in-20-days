### Q1. Child-Pugh score — components, grading and use (5m)
<!-- id: sys06-crit-q1 -->

**Restated:** Describe the Child-Pugh score for grading severity of cirrhosis.

- **Purpose (1m):** grades **severity / prognosis of chronic liver disease (cirrhosis)** and historically
  predicts operative/surgical risk. Five variables, each scored **1–3**; total **5–15**. (1m)
- **The five components (2m):**

| Variable | 1 point | 2 points | 3 points |
|---|---|---|---|
| **Bilirubin** (mg/dL) | <2 | 2–3 | >3 |
| **Albumin** (g/dL) | >3.5 | 2.8–3.5 | <2.8 |
| **INR (PT)** | <1.7 | 1.7–2.3 | >2.3 |
| **Ascites** | none | mild (controlled) | moderate–severe (refractory) |
| **Encephalopathy** | none | grade I–II | grade III–IV |

  (Mnemonic **"ABCDE"** — **A**lbumin, **B**ilirubin, **C**oagulation/INR, **D**istension/ascites,
  **E**ncephalopathy.) (2m for the full table)

- **Grading + use (1½m):** **Class A = 5–6** (well-compensated), **Class B = 7–9** (significant functional
  compromise), **Class C = 10–15** (decompensated). Used for prognosis, transplant assessment and
  pre-operative risk. (1½m)
- **Worked example (½m):** bilirubin 2.5 (2) + albumin 2.5 (3) + INR 2.0 (2) + mild ascites (2) +
  grade I encephalopathy (2) = **11 → Class C**. (½m)

### Q2. MELD score — components, formula and use (5m)
<!-- id: sys06-crit-q2 -->

**Restated:** Describe the MELD score and its use in liver disease.

- **Purpose (1m):** **Model for End-stage Liver Disease** — an objective score predicting **3-month
  mortality** in chronic liver disease; the basis for **prioritising liver-transplant allocation**. (1m)
- **Components (1½m):** three **objective** lab values — **serum bilirubin, serum creatinine, INR**
  (the original MELD). **MELD-Na** adds **serum sodium**. No subjective variables (unlike Child-Pugh). (1½m)
- **Formula (1m):**
  - MELD = **3.78 × ln[bilirubin mg/dL] + 11.2 × ln[INR] + 9.57 × ln[creatinine mg/dL] + 6.43**.
  - Lab values **<1.0 are set to 1.0** (ln 1 = 0, avoids negatives); range capped (≈6–40). (1m)
- **Use / interpretation (1m):** higher score = higher mortality and **higher transplant priority**;
  e.g. **MELD ≥15** is the usual threshold at which transplant benefit outweighs risk. Recalculated
  periodically on the waiting list. (1m)
- **Worked example (½m):** bilirubin 1, INR 1, creatinine 1 → all ln terms = 0 → MELD = **6.43 ≈ 6**
  (the practical floor for a stable patient). (½m)

### Q3. Ranson's & modified Glasgow criteria for acute pancreatitis severity (5m)
<!-- id: sys06-crit-q3 -->

**Restated:** Outline the criteria used to assess severity in acute pancreatitis.

- **Purpose (1m):** predict **severity / mortality** in acute pancreatitis to triage to HDU/ICU; severity
  rises with the number of positive criteria. (1m)
- **Ranson's criteria (2m)** — gallstone-pancreatitis values noted differ slightly:

| **On admission** | **Within 48 hours** |
|---|---|
| Age >55 yr | Haematocrit fall >10% |
| WBC >16,000/mm³ | Urea (BUN) rise >5 mg/dL |
| Glucose >200 mg/dL | Calcium <8 mg/dL |
| LDH >350 IU/L | PaO₂ <60 mmHg |
| AST >250 IU/L | Base deficit >4 mEq/L |
|  | Fluid sequestration >6 L |

  (Mnemonic admission **"GA LAW"**; 48-h **"Calvin & HOBBS"**.) (2m for the table)

- **Modified Glasgow (Imrie) — "PANCREAS" (1m):** **P**aO₂ <60 mmHg · **A**ge >55 · **N**eutrophils
  (WBC) >15,000 · **C**alcium <8 mg/dL · **R**enal urea >16 mmol/L · **E**nzymes LDH >600 · **A**lbumin
  <32 g/L · **S**ugar (glucose) >180 mg/dL — assessed within 48 h. (1m)
- **Interpretation (1m):** **≥3 criteria = severe / predicted complicated course**; Ranson ≥3 (or APACHE-II
  ≥8, CRP >150 at 48 h) predicts severe disease and higher mortality. Limitation: Ranson needs **full 48 h**
  to complete. (1m)
